🔍 根因分析

为什么会出现这些偏离？

  1. Phase 3 实施时：直接用 pi-web fork 覆盖了 packages/ui，没有按照设计文档 §4.1 重新组织目录结构（缺失 src/ 层级）
  2. 工具实现重复：bioagent-tools.ts 独立 import dockerode 实现 docker 工具，而不是通过 pi 的 tool mechanism 调用 executor 包的 API
  3. API 路由未对接：workflow/knowledge/files API 路由存在但 handler 未调用 backend 包的对应方法
  4. 设计文档未及时同步：部分设计细节在实际开发中发现不适用但未更新设计文档
