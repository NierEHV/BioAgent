# 案例：批次效应误诊为生物学差异

## 场景

某研究比较了治疗组和对照组的 scRNA-seq 数据。治疗组样本在 A 平台上测序，
对照组在 B 平台上测序。分析发现两组之间有明显差异，研究者将这些差异解释
为治疗效果。

## 问题

- 治疗条件与测序平台**完全混淆**
- PCA 显示第一主成分由"批次"解释（96% 方差）
- 差异表达基因中包含大量核糖体和线粒体基因

## 诊断方法

1. **按批次着色 PCA**: 观察是否按技术因素分离
2. **检查 top DE genes**: 是否富集核糖体/线粒体基因
3. **计算 k-BET**: 批次混合程度
4. **与已知 marker 对比**: 差异基因是否包含预期的生物学 marker

## 修复方案

1. **批次校正**: Harmony 或 scVI 整合数据
2. **保留分析**: 仅在每个批次内部分别分析，取交集
3. **实验设计**: 下次将治疗/对照分散到多个批次

## 教训

- ⚠️ 实验设计阶段就要避免生物条件与技术批次的完全混淆
- ⚠️ PCA 是批次效应的第一道检测工具
- ⚠️ 差异基因列表需要生物学合理性审查
- ⚠️ 核糖体/线粒体基因在 DE 列表中富集是典型的批次效应信号
